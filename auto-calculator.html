Задание 3

Задачи с массивами являются одним из часто встречаемых в работе программиста и, как следствие, на собеседованиях. Поэтому давай потренируемся использовать изученные методы.
  1. Дан массив ['js', 'css', 'html']. Выведите на экран первый элемент.

    2. let testArray_1 = ["js", "css", "html"];
    console.log(testArray_1[0]); //js

  2. Отфильтруйте массив [0, 1, false, 2, undefined, '', 3, null] от нежелательных значений, таких как false, undefined, пустые строки, 0, null с помощью метода filter. Ожидаемый результат: [1, 2, 3].

    3. const filteredTestArray_2 = testArray_2.filter(Boolean);
    console.log(filteredTestArray_2); //[1, 2, 3]


  3. Дан массив [[1,2], [1,2,3], [1,2,3,4]]. Найдите индекс массива, длина которого > 3. Ожидаемый результат: 2

    let testArray_3 = [ [1, 2], [1, 2, 3], [1, 2, 3, 4], ];
    let filteredTestArray_3 = testArray_3.filter((item) => item.length > 3);
    console.log(filteredTestArray_3); // [1, 2, 3, 4]
